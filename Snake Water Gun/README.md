🐍 Snake Water Gun

A simple command-line game similar to Rock-Paper-Scissors where:

Snake 🐍 drinks Water 💧

Gun 🔫 kills Snake 🐍

Water 💧 destroys Gun 🔫

The computer randomly chooses one, and you compete against it!

🚀 How to Run

python snake_water_gun.py

📚 Concepts Used

1)Random module

2)Conditional statements

3)Loops
